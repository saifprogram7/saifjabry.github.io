About The Software

Jarvis is a Voice Assistant designed which is designed with two things in mind User Friendly and Efficiency. The software has 17 features which include the following:

- Calculator
- Command Line
- Discord
- Search
- Chrome
- VS Code
- Video
- Website
- Date
- Time
- Folder
- File
- Word
- OBS
- PowerPoint

All of these features are customizable, meaning that you can add new features or make some changes to the features based on the laptop / computer you are using. I will be providing a documentation of how you can make the changes within the software if you require.

The software has Voice Recognition System installed within the software, this allows you to talk to the software and it will execute a function based on your voice input. This is a unique feature which allows you to talk into the software if you have any injuries which restricts you from using the device and also it allows you to interact with Jarvis the same way Tony Stark talks with his personal voice assistant Jarvis.
